Gilt Style Guide

Basics

As our final project in my Web UI Development Accelerator we were to make a style guide. A style guide is a resource for developers of a particular brand or site that contains guidelines and the code to build that particular site. Code for America and Primer (for GitHub) were used as exmaples of real style guides. I chose to do a style guide for Gilt.

My Role

This was an individual project. I picked the website to make a style guide for, Gilt, and planned out I'd break down the different style components. Keeping in mind that the style guide site and the styles for Gilt are really two different sites, the style guide being an application within the style guide, was important to keep in mind throughout the process of building the style guide.

Technologies

Within Node and Express, I used HTML, Sass, and some JavaScript to build this site. I installed my Bower package (Saskatoon) on this project because it has a well-organized (inspired by MVC) file structure for Sass. Grid Systems were a top priority when working on this because I needed a different grid system for my style guide's site and for Gilt's site. I also used Prism to display my code for the style guide. .
